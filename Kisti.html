치매 예방 및 관리 프로그램 6명 - Client/PM

카이스트 산학협력 사업으로 개발한 프로젝트
환자의 VR 게임데이터를 통해 치매 기능 측정 데이터를 받아오고 관리
받아온 데이터를 런처를 통해 교수, 보호자가 확인 할 수 있는 기능 제공
화상채팅 기능을 통해 원격으로 VR게임 치매 예방 교육을 진행하는 기능 제공

개발 기간

  • 2023.05 - 2023.11 (7개월)

개발 도구

  • Unity, C#
  • Photon
  • Agora API

개발 내용

  • Agora API를 이용하여 화상채팅 기능 개발
  • Photon을 이용한 VR 게임 사용자 게임 제어
  • 사용자 정보 캡처 후 PDF파일로 저장 기능 개발
  • 전제적인 런처 기능 및 UI 개발
